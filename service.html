Shadow Sketch

Shadow sketch is one of my all time favorite technique because it is easier to make. I usually take only 30-45 minutes to finish a single person sketch. This technique can be done by recognized the shadow on the face or body and basically it only use black color. Some creative artists are using different type of colors like dark gray, light gray and more to make the sketch more detail.

Colored Sketch

Just like the other colored sketch, I implement more color to match with the tone of the skin color and some rough detailing to make it real. The result is like an animation sketch and a bit 3D because of the shadow and light from some spot.
